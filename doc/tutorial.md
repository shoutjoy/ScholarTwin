# ScholarTwin 사용 튜토리얼

이 문서는 현재 앱에 있는 **메뉴 기반 기능 전체**를 빠르게 익히기 위한 가이드입니다.

## 1. 시작하기

### 1-1) 로그인
- 앱 실행 후 로그인 화면(`AuthModal`)에서 계정으로 로그인합니다.
- 로그인 후 상단 헤더와 메인 화면이 표시됩니다.

### 1-2) 파일 업로드
- 중앙 업로드 영역에서 파일을 드래그 앤 드롭하거나 클릭해서 선택합니다.
- 지원 형식:
  - `PDF`
  - `STP` 프로젝트 파일(`.stp`)
- PDF 업로드 시:
  - 페이지 수 확인
  - 원문 텍스트 추출(페이지 단위)
- STP 업로드 시:
  - 저장된 프로젝트 상태를 복원합니다.

---

## 2. 상단 헤더 메뉴 사용법

### 2-1) 작업 진입 메뉴
- `원문번역`
  - 좌측 원문 / 우측 AI페이지번역의 페이지 단위 비교 화면으로 이동
- `원문+번역 페이지`
  - TwinView(원문/번역 쌍뷰 + PDF/이미지/북마크) 화면으로 이동
- `Prev`
  - 시작(업로드) 화면으로 돌아갑니다.

### 2-2) 번역 실행 메뉴
- `[페이지번역]`
  - 원문번역 화면에서 페이지 단위 AI 번역 실행
  - 진행 중에는 `[페이지번역 중]` 상태로 표시
- `[차례로]`
  - 기본 페이지 범위(예: 1-2) 순차 번역/분석 실행
- `[full시작]`
  - 전체 범위 번역/분석 실행

### 2-3) 결과 도구 메뉴
- `Study Assistant`
  - 우측 도구 패널 열기
- `PDF ▼`
  - PDF 폰트 확대/축소
  - `새창 PDF`
  - 원본/번역 PDF 다운로드
- `TXT ▼`
  - `English Only`
  - `Korean Only`
  - `Twin View`
  - `Prepare for NotebookLM`

### 2-4) 세션/설정/기타 메뉴
- `세션`
  - 우측 저장/불러오기 사이드 패널 열기/닫기
- `설정(톱니바퀴)`
  - AI 설정 모달 열기
- `채팅 아이콘`
  - 번역 결과가 있을 때 문서 채팅 패널 열기
- `초기화`
  - 현재 번역 결과/상태 초기화
- `Logout`
  - 로그아웃

---

## 3. 시작 화면(파일 업로드 이후) 메뉴

PDF 업로드 후 아직 번역 화면에 들어가기 전, 좌우 준비 화면에서 아래 메뉴를 사용합니다.

### 3-1) 좌측 탭
- `PDF`
  - PDF 미리보기
  - `Open Synced PDF Window`로 외부 PDF 창 열기
- `원본(text)`
  - 추출된 원문 텍스트 확인(페이지 마커 포함)
- `오픈소스 번역결과`
  - LibreTranslate 결과 확인
  - 페이지별 `Google번역으로 보내기`
  - 전체 원문 Google 번역 열기

### 3-2) 우측 설정
- `Translation Style`
  - `Academic` / `Explanatory`
- `Extraction Range`
  - `입장하기`
  - 페이지 범위 입력(`e.g. 1-2`)
  - `[차례로]`, `[Full extract]`
- `Cancel & Upload Different File`
  - 다른 파일로 교체

---

## 4. 원문번역 화면 사용법

경로: 상단 `원문번역` 버튼 클릭

### 4-1) 기본 구조
- 좌측: `원문` (페이지 블록)
- 우측: `AI페이지번역` (페이지 블록)
- 페이지 단위 높이 보정 + 스크롤 동기화로 비교 가독성 향상

### 4-2) 주요 기능
- 우측 상단 `Sync ON/OFF`
  - 원문/번역 스크롤 동기화 켜기/끄기
- 상단 `[페이지번역]`
  - AI 페이지 번역 실행
- 진행 상태
  - 상단 진행률(%) + 상태 메시지 표시

---

## 5. TwinView 화면 사용법

경로: 상단 `원문+번역 페이지` 버튼 클릭

### 5-1) 좌측 패널
- `원문 text`
  - 원문 보기 설정 팝업 열기
- 모드 탭:
  - `Text`
  - `Figs/Tables`
  - `★`(북마크)
  - `PDF`
- PDF 모드 부가 기능:
  - `- / % / +` 줌
  - `새창 PDF`

### 5-2) 원문 보기 설정(팝업) 기능
- 페이지 확대/축소: `페이지 - / % / +`
- 폰트 확대/축소: `폰트 - / % / +`
- `스크롤 동기화 ON/OFF`
- `새창` (확대/축소/인쇄 포함)
- `전체화면` / `닫기`
- 내용 모드:
  - `영어만`
  - `한국어만`
  - `함께 보기`
- `내용복사`
  - 현재 미리보기 텍스트를 클립보드 복사

### 5-3) 우측 패널
- `Translated Twin`
  - 번역 세그먼트 확인

### 5-4) 하단 추가 로딩
- `Load Next 1 Page`
  - 다음 페이지를 1페이지 단위로 추가 로딩

---

## 6. 왼쪽 네비게이션 사이드바

### 6-1) 공통 기능
- 펼침/접힘 토글 버튼
- 탭:
  - `Pages`
  - `Outline`

### 6-2) Pages 탭
- 페이지 번호 클릭으로 이동
- 처리 완료 페이지는 상태 점(녹색)으로 표시

### 6-3) Outline 탭
- 감지된 제목(heading) 목록 표시
- 항목 클릭 시 해당 세그먼트로 이동

---

## 7. Study Assistant 패널

경로: 상단 `Study Assistant`

### 7-1) 공통
- 상단 탭:
  - `Explore`
  - `Conclusion`
  - `PPT Script`
- 생성결과 저장/불러오기:
  - `생성결과 저장` (IndexedDB)
  - `생성결과 불러오기`

### 7-2) Explore 탭
- `Explore theory` 버튼으로 핵심 용어 추출
- 용어 카드에서:
  - 영문/한국어 용어, 정의, 문맥
  - `Deep Explain with AI`
- `Download TXT`

### 7-3) Conclusion 탭
- `Generate Conclusion`
- 항목:
  - Research Questions
  - Key Results
  - Implications
- 영문+한국어 병기 지원
- `Download TXT`

### 7-4) PPT Script 탭
- `Create PPT Script`
- 언어 전환:
  - `영문`
  - `한국어`
- 다운로드:
  - `Download .txt`
  - `한국어 .txt`
- `Open in New Window`
  - 새창에서 확대/축소/복사/인쇄
- 스크립트 박스는 세로 리사이즈 가능

---

## 8. 세션(저장/불러오기) 패널

경로: 상단 `세션`

### 8-1) 상단 버튼
- `현재 상태 IndexedDB 저장`
- `저장 목록 새로고침`
- `.stp 프로젝트 파일 내보내기`
- `.stp 프로젝트 파일 불러오기`

### 8-2) 저장 목록
- 세션 카드별:
  - `불러오기`
  - `삭제`

---

## 9. 설정(톱니바퀴) 메뉴

### 9-1) 원문 추출 방식
- `PDF 텍스트 레이어(기본)`
- `AI 이미지 분석`

### 9-2) AI Provider
- `Google Gemini (AI Studio)`
- `External API (ChatGPT etc.)`

### 9-3) 모델/키 설정
- 텍스트 모델
- 이미지 모델
- API Key 입력/표시 토글

### 9-4) 저장
- `Save Configuration` 클릭 시 설정 저장

---

## 10. 권장 사용 순서(빠른 워크플로우)

1. 파일 업로드(`PDF` 또는 `.stp`)
2. 필요 시 `설정`에서 추출 방식/API 확인
3. `원문번역` 또는 `원문+번역 페이지` 진입
4. `[페이지번역]` 또는 `[차례로] / [full시작]` 실행
5. `Study Assistant`에서 Explore/Conclusion/PPT 생성
6. `PDF/TXT` 메뉴로 결과 내보내기
7. `세션`에서 IndexedDB 저장 또는 `.stp` 내보내기
